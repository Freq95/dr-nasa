# CMArt Dent

Landing page pentru clinica stomatologica CMArt Dent, setata ca proiect Vite + React + TypeScript + Tailwind.

## Run

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build -> dist/
npm run preview  # preview the build
```

## Theme

Componenta foloseste token-uri semantice de tip shadcn (`bg-primary`, `text-muted-foreground`, etc.), iar paleta neutra OKLCH este definita in `src/index.css` si `tailwind.config.js`.
